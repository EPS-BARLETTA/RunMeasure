RunMeasure Pro — Pack complet
--------------------------------

Ce pack implémente :
- Intermédiaires avec distance totale + pas (auto-stop au but) + QR
- Chrono plein écran
- Minuteur plein écran avec compte à rebours circulaire + roues H/M/S
- Chrono avec vitesse (fullscreen + calcul en direct)
- Cooper & Demi-Cooper : longueur de piste + tours + fraction 1/4, 1/2, 3/4
- Minuteur avec distance : mêmes réglages de piste pour plus de précision
- QR JSON compatible (app, mode, élève, result)

Déployez tout à la racine de GitHub Pages. Ouvrez options.html.
